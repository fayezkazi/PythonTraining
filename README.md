## ğŸš€ Getting Started

### Prerequisites

- Python 3.6 or higher installed on your system
- A text editor or IDE (VS Code, PyCharm, Sublime Text, etc.)
- Basic understanding of computer programming (for beginners)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/fayezkazi/PythonTraining.git
cd PythonTraining
```

2. (Optional) Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

## ğŸ“ Exercises

### Level 1: Basics

**01_variables.py** - Learn about:
- Creating and using variables
- Basic data types (string, int, float, bool)
- Type checking
- String formatting

**02_control_flow.py** - Practice:
- If/elif/else statements
- For loops and while loops
- Nested loops
- Break and continue statements
- FizzBuzz challenge

**03_data_structures.py** - Work with:
- Lists and list operations
- Tuples and immutability
- Dictionaries and key-value pairs
- Sets and uniqueness
- List comprehensions

### Level 2: Intermediate

**01_functions.py** - Master:
- Function definitions and calls
- Parameters and return values
- Default parameters
- Lambda functions
- Map, filter, and reduce

**02_oop.py** - Understand:
- Classes and objects
- Constructors (__init__)
- Instance methods and attributes
- Inheritance and polymorphism
- Class methods and static methods
- Magic methods (dunder methods)

**03_file_io.py** - Learn:
- Reading and writing files
- Exception handling (try/except/finally)
- Context managers (with statement)
- Working with CSV files

### Level 3: Advanced

**01_decorators.py** - Explore:
- Basic decorators
- Decorators with arguments
- Chaining decorators
- Class decorators
- Common decorator patterns

**02_generators.py** - Study:
- Generator functions and yield
- Generator expressions
- Custom iterators
- Infinite sequences
- Memory-efficient iteration

**03_context_managers.py** - Implement:
- Context manager protocol
- Using contextlib
- Custom context managers
- Resource management
- Exception handling in context managers

## ğŸ“– How to Use

### For Self-Study

1. **Start with the basics** (if you're new to Python)
2. **Read the exercise file** and understand what's required
3. **Try to solve the exercise** without looking at the solution
4. **Run your code** to test it:
   ```bash
   python exercises/01_basics/01_variables.py
   ```
5. **Check the solution** if you get stuck or want to compare approaches
6. **Experiment** by modifying the code and seeing what happens

### For Instructors

- Use exercises as homework assignments
- Solutions can serve as answer keys
- Modify exercises to suit your teaching needs
- Add your own exercises following the same structure

### Best Practices

- **Write code**, don't just read it
- **Experiment** with variations of each exercise
- **Break things** - errors are learning opportunities
- **Comment your code** to explain your thinking
- **Practice regularly** - consistency is key
